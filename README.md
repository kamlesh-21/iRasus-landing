# Preksha Battery Monitoring System - Demo Site

## Project Overview
Landing page for booking demos of Preksha Battery Monitoring System. Target audience: EV dealers, fleet operators, and battery assemblers in India.

## Technology Stack
- HTML5 (Semantic, SEO-optimized)
- CSS3 (Custom properties, responsive design)
- Vanilla JavaScript (No frameworks for performance)
- Progressive Web App (PWA) ready

## SEO Implementation
- Complete meta tags (Open Graph, Twitter Cards)
- Schema.org structured data (FAQ, SoftwareApplication, Organization)
- XML sitemap with multilingual support
- robots.txt with crawl directives
- humans.txt and llms.txt for transparency
- Canonical URLs and hreflang tags

## Performance Features
- Lazy loading (images/scripts)
- CSS/JS minification (in production)
- Font display optimization
- Critical CSS inlined
- Asset preloading
- HTTP/2 ready structure

## Accessibility
- WCAG 2.1 AA compliant
- ARIA labels and roles
- Keyboard navigation support
- Screen reader optimized
- Color contrast compliant
